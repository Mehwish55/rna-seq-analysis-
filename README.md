# RNA-seq Analysis (TCGA Dataset)

## 🧬 Objective
Perform differential gene expression analysis on cancer RNA-seq data.

## 📊 Dataset
Source: TCGA (The Cancer Genome Atlas)

## ⚙️ Tools & Technologies
- R (DESeq2)
- Bioconductor
- Linux

## 🔬 Workflow
1. Data collection
2. Data preprocessing
3. Normalization
4. Differential expression analysis
5. Visualization (heatmaps, volcano plots)

## 📈 Expected Results
- List of differentially expressed genes (DEGs)
- Volcano plot
- Heatmap

## 🚀 Status
Project in progress – setting up analysis pipeline
